Battleship это браузерная игра Морской бой с современным интерфейсом.


Функции:

- Одиночная игра против бота с тремя уровнями сложности

- Настраиваемое размещение кораблей перед боем (вручную или авторасстановка)

- Мультиплеер в реальном времени: два игрока играют по ссылке-приглашению, с установленной системой часов как в шахматах

- Уникальные режимы игры для 3 и 4 игроков по сети

- Глобальный рейтинг (только победы над ботом) и личная статистика матчей для авторизованных пользователей

- Звуковые эффекты, 3D-доска, маркировка клеток правой кнопкой мыши


Для кого:
Для людей кто хочет сыграть в Морской бой с другом онлайн без регистрации и против ботов.

Это ценно потому что мультиплеер работает полностью в браузере без установки чего-либо, 
задержки минимальны, а авторизация позволяет сохранять историю матчей и видеть свой прогресс в таблице лидеров.
